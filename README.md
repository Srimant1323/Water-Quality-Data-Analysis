# Water Quality Data Analysis (EDA)

This is a self-practice data analysis project exploring water quality parameters and their relationship with water potability.

The goal of this project is to perform Exploratory Data Analysis (EDA) and visualize relationships between different water quality parameters.

---

## Dataset

The dataset contains various chemical properties of water used to determine whether the water is potable (safe to drink).

Features include:

- pH
- Hardness
- Solids
- Chloramines
- Sulfate
- Conductivity
- Organic Carbon
- Trihalomethanes
- Turbidity
- Potability

---

## Project Workflow

The following steps were performed in this analysis:

1. Data Loading
2. Data Cleaning
3. Handling Missing Values
4. Exploratory Data Analysis (EDA)
5. Correlation Analysis
6. Data Visualization

---

## Visualizations

The project includes multiple visualizations such as:

- Correlation Heatmap
- Distribution plots
- Feature analysis graphs

These visualizations help understand relationships between water quality parameters.
